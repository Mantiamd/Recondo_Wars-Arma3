RECONDO WARS - INTEL PHOTOS
============================

This folder contains default images for the Intel system.

Required File:
--------------
default_photo.paa - Default silhouette photo (256x256)
                    Used when no specific photo is provided for HVT/Hostages

To create default_photo.paa:
1. Create a 256x256 pixel image (silhouette of a person)
2. Convert to .paa using Arma 3 Tools (TexView2 or ImageToPAA)
3. Place the file in this directory

Mission makers can provide their own photos in the mission folder.
Suggested mission folder structure:
  mission_folder\photos\hvt_name.paa
  mission_folder\photos\hostage_name.paa

Supported formats: .paa (recommended), .jpg, .png
