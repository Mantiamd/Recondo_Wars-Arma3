RECONDO WARS - AMBIENT WILDLIFE SOUNDS
======================================

This folder contains default ambient wildlife sounds for the
Ambient Sound Triggers module.

Required Files:
---------------
- monkeys_1.ogg
- monkeys_2.ogg
- monkeys_howling.ogg

Sound Specifications:
--------------------
- Format: OGG Vorbis (.ogg)
- Sample Rate: 44100 Hz recommended
- Channels: Mono recommended for 3D positioning
- Duration: 2-10 seconds typical

Adding Custom Sounds:
--------------------
Mission makers can use custom sounds by:
1. Placing .ogg files in their mission folder
2. Setting "Custom Sounds Path" in the module (e.g., "sounds\ambient")
3. Setting "Custom Sounds List" with filenames (e.g., "custom1.ogg, custom2.ogg")

Future Categories:
-----------------
Additional sound categories will be added in future updates:
- dogs (barking, growling)
- birds (startled, calls)
- insects (cicadas, night sounds)
- generic (rustling, branches)
